# Jodit NodeJS Connector
Upload and file browser connector for Jodit editor

# Get started

- `npm install -g nodemon`
- `npm install`
- Configure your `.env` files (_see `.env-sample`_)
- `nodemon ./server.js`


Happily written in Hapi.
